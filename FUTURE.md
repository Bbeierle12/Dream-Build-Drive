# FUTURE

Backlog items that are intentionally out of scope for the active phase. Nothing in this file is committed work until it is promoted into the roadmap.

## Future Product Areas

- Collaboration and sharing
  - multi-user task assignment with role-based access
  - live presence indicators and real-time updates
  - public read-only project sharing
- Notifications and automation
  - due date reminders by email or push
  - blocker alerts for overdue dependencies
  - recurring maintenance templates
- Mobile and field use
  - responsive garage-floor optimization
  - offline-first sync for weak connectivity
  - quick camera capture for part photos
- Vehicle intelligence
  - OBD-II diagnostics integration
  - AI-assisted cost estimation
  - vehicle database integration for part numbers and specs
- Data portability
  - CSV and PDF export expansions
  - project backup and restore
  - spreadsheet import
- Monetization, only if product value is proven
  - Stripe subscription tiers
  - multi-tenant workspace architecture

## Research Backlog

- assess whether offline-first materially improves the shop-floor workflow
- evaluate vehicle database APIs for specs and part lookups
- define the usefulness threshold for AI-assisted cost estimation
- map a migration path from single-user to multi-tenant collaboration
- review how competing automotive project trackers store and present specs
